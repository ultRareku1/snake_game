# 贪吃蛇 · Snake

一个现代极简风格的贪吃蛇网页游戏。单文件实现，零依赖，打开即玩。

## 功能特性

- **现代极简 UI**：圆角棋盘、细腻网格、柔和发光效果，视觉干净克制
- **明暗双主题**：一键切换暗色 / 亮色主题，自动跟随系统偏好，选择持久保存
- **平滑动画**：蛇身移动采用逐帧插值渲染，告别传统格子跳动感
- **三档难度**：悠闲（150ms）/ 标准（105ms）/ 极速（72ms），且随进食逐渐加速
- **音效反馈**：WebAudio 实时合成的轻量音效，可一键静音
- **最高分记录**：使用 localStorage 本地保存最高分，达成新纪录时特别提示
- **全平台操作**：桌面键盘 + 移动端滑动手势 / 虚拟方向键
- **细节处理**：切后台自动暂停、分数跳动动画、180° 转向保护、高 DPI 屏幕适配

## 快速开始

无需构建、无需安装依赖，直接用浏览器打开 `index.html` 即可：

```bash
# 方式一：直接双击 index.html

# 方式二：本地起一个静态服务器（推荐）
cd snake-game
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 操作说明

| 操作 | 桌面端 | 移动端 |
| --- | --- | --- |
| 移动 | `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 滑动屏幕或点击方向键 |
| 开始 / 重开 | 任意方向键或 `回车` | 轻触棋盘 |
| 暂停 / 继续 | `空格` | — |

规则：吃到食物 +10 分；撞到墙壁或自己的身体则游戏结束。

## 技术实现

- **单文件架构**：HTML + CSS + 原生 JavaScript 全部内联于 `index.html`，无任何外部依赖
- **渲染**：Canvas 2D + `requestAnimationFrame`，逻辑帧（tick）与渲染帧分离，通过线性插值实现平滑移动
- **主题系统**：CSS 自定义属性（变量）驱动，`data-theme` 切换，画布配色随主题联动
- **音效**：WebAudio API 振荡器实时合成，无音频资源文件
- **持久化**：`localStorage` 保存最高分、主题、难度与静音偏好
- **自适应**：`ResizeObserver` + `devicePixelRatio` 缩放，保证任意屏幕下画面锐利

## 自定义

游戏核心参数集中在 `index.html` 底部脚本的常量区，可直接修改：

| 参数 | 位置 | 说明 |
| --- | --- | --- |
| `GRID` | 脚本顶部常量 | 棋盘格子数（默认 21 × 21） |
| `data-tick` | 难度按钮 HTML | 三档难度的基础速度（毫秒，越小越快） |
| CSS 变量 | `:root` / `[data-theme="light"]` | 全部配色、圆角、阴影 |

## 浏览器兼容

支持所有现代浏览器（Chrome、Edge、Firefox、Safari 最新版）及移动端浏览器。主题联动使用了 `color-mix()`，旧版浏览器会优雅降级。
